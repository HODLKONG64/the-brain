## IMAGE GENERATION & ON-BRAND CONSISTENCY RULES

### Core Principle
Images must represent characters faithfully according to the established definitions, ensuring visual consistency across all outputs.

### Primary Reference Sources
Use the following reference sources for image generation:
- Official concept art
- Character designs from approved games or media
- Style guides that define character aesthetics

### If NO Reference Images Can Be Found (fallback rules)
- Use speculative design based on character traits and lore.
- Prioritize visual elements that are most prominent in the character's description.

### How the Agent Ties Images to the Final Lore Details
Images must be cross-referenced with the character's final lore details to maintain narrative coherence. This includes:
- Aligning appearance with character backstories.
- Ensuring visual cues correspond with character development arcs.

### Enforcement & Consistency Checks
Regular audits must be performed to ensure adherence to these guidelines:
- Peer reviews for image submissions.
- Feedback loops with character designers and writers to address inconsistencies.

### Why These Rules
These rules ensure that the brand's imagery remains consistent, enhancing audience recognition and engagement with the characters and their stories.
