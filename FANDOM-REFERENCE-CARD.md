# Fandom Reference Card

## Quick Reference Guide

- **Fandom Overview:** A platform for fans to create and share content about their favorite topics.
- **Creating a Page:**  To create a new page, navigate to the wiki, find the desired title, and click "Create Page."
- **Editing Pages:**  Click on the "Edit" button on the page you want to edit to make changes.
- **Adding Media:** Use the "Upload" button to add images or media files.
- **Formatting Tips:**  Use markdown syntax for headings, links, and lists.
- **Community Guidelines:**  Be respectful and follow the community's rules.
